# Streamlit Frontend Application

This directory contains the user interface for the Car Damage Classifier. 
It loads the trained MobileNetV3 model to process uploaded images and display vehicle damage predictions.

## Prerequisites

Before running the application, ensure you have installed the required libraries listed in `requirements.txt`:

```bash
pip install -r requirements.txt
```

## Local Execution

To run this application independently, navigate to this (streamlit_app) directory and execute:

```bash
streamlit run app.py
```
